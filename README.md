# 💳 Synthetic Credit Card Fraud Detection (Streamlit App)

This project generates synthetic credit card transaction data with a customizable fraud rate and runs fraud detection models like Logistic Regression and Random Forest.

## 🚀 Features
- Generate 1,000 – 200,000 synthetic transactions
- Set fraud rate (0.1% – 10%)
- Preview dataset in browser
- Train & evaluate models
- View classification report & confusion matrix
- Download dataset as CSV

## 📦 Installation
```bash
git clone https://github.com/yourusername/credit-card-fraud-streamlit.git
cd credit-card-fraud-streamlit
pip install -r requirements.txt
```

## ▶ Run Locally
```bash
streamlit run app.py
```

## ☁ Deploy to Streamlit Cloud
1. Push your repo to GitHub
2. Go to Streamlit Cloud
3. Connect your repo & deploy
